### Hey PH! 👋

I built AutoTask because browser automation shouldn't require a DevOps degree.

**The Problem:**
- Zapier/Make are powerful but expensive ($25-75/month gets you... nothing useful)
- n8n requires hosting and maintenance
- Browser automation tools are either too simple or too complex

**The Solution:**
AutoTask runs entirely in your browser with a Chrome extension + Cloudflare Worker backend. No servers to manage, no complex setup.

**What You Can Do (MVP):**
1. **Lead Scraping** - Extract leads from LinkedIn, Crunchbase, company websites
2. **Form Filling** - Auto-fill contact forms, registrations, surveys
3. **Data Extraction** - Pull structured data from product pages, articles

**How It Works:**
1. Install Chrome extension
2. Pick a workflow template (or create custom)
3. Run automation - AI handles the browser interaction
4. Results saved to your dashboard

**Tech Stack:**
- Next.js 15 web dashboard
- Chrome Extension (Manifest V3)
- Cloudflare Workers (zero-infrastructure backend)
- Browserbase integration for headless browser automation

**Pricing (Launch):**
- Free tier: 10 workflows/month
- Solo: $15/month (500 workflows)
- Team: $49/month (unlimited)

**What's Next:**
- Supabase integration for persistent storage
- OAuth connections to CRM/email tools
- More workflow templates
- AI-enhanced action planning
