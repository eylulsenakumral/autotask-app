🔐 Privacy Policy

Last Updated: January 2026

AutoTask is designed with privacy as the core principle. We believe automation tools should respect your data, not exploit it.

Data Collection ZERO

  • Zero data collection — we collect nothing
  • No analytics, no tracking, no user accounts required
  • No cloud servers — all automation runs locally on your device

Data Storage

  • Workflows stored locally in your browser's extension storage
  • Optional encrypted cloud backup (user-enabled, enterprise only)
  • No personally identifiable information is collected or transmitted

Permissions Required

AutoTask requires the following Chrome permissions to function:

  • activeTab — To record and replay actions on the current page
  • storage — To save your workflows locally
  • host permissions — Only for domains you explicitly authorize
  • scripting — To inject automation scripts into pages you control
  • tabs — To manage multiple browser windows during workflows

Data Sharing

  • We do not sell, rent, or share any data
  • No third-party services or SDKs
  • No advertising or data monetization

Data Retention

  • You control your data — delete anytime
  • Uninstalling removes all local data
  • Export workflows before uninstalling for backup

Security

  • All data encrypted at rest (Chrome standard)
  • No remote code execution
  • Open source code available for audit

Contact

For privacy concerns: autotask@example.com

This privacy policy applies to the AutoTask Chrome extension. By using AutoTask, you agree to this policy.
